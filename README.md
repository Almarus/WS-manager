WS Manager — бесплатное программное обеспечение для безопасного отключения служб в Windows.

## 🚀 Установка и запуск

1. Скачайте последнюю версию [Releases](https://github.com/Almarus/Video-Grabber/releases)
2. Запустите `WS Manager.exe` от имени администратора
3. Дождитесь окончания сканирования системы
4. Отключите через интерфейс ПО или контекстное меню ненужные службы
5. Наслаждайтесь оптимизированной ОС
