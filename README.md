WS Manager — бесплатное программное обеспечение, предназначенное для безопасного отключения системных служб Windows, не востребованных большинством пользователей. Проект ориентирован на оптимизацию операционной системы, снижение фоновой нагрузки на аппаратные ресурсы (CPU/RAM) и повышение общей производительности ПК без риска что-то сломать. Содержит множество встроенных систем от "дурака". Поддерживает Windows 10/11.
